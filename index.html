Tentang Saya 𝜗ৎ

Halo, saya Yohana, mahasiswa Teknologi Rekayasa Perangkat Lunak yang tertarik mendalami UI Design dan pengembangan web. Saya sering menggunakan HTML dan CSS untuk mengerjakan berbagai tugas, seperti membuat website berita, website bisnis untuk kelas entrepreneur (Design Lab), serta login form sederhana.

Selain itu, saya juga mencoba membuat website untuk menampilkan hasil karya saya seperti lukisan dan hobi lainnya. Saya tertarik belajar menggunakan Figma dan cukup mahir memakai Canva untuk desain antarmuka. Saat ini saya masih mempelajari dasar Java dan Python, serta menggunakan GitHub dan media sosial untuk membagikan project dasar yang saya kerjakan.

Tujuan saya adalah terus mengembangkan kemampuan di bidang web development dan UI/UX design, sambil membangun portofolio karya yang bisa menjadi langkah awal menuju karier sebagai software engineer.
